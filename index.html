名簿作成ツール (management)
警備員名簿の作成とプロキャスデータ更新を自動化するツール集。Excelテンプレートを使用して名簿を自動生成し、写真の自動貼り付け、データ更新、ログ管理機能を提供。バッチ処理対応で複数名簿の一括作成が可能。
主な機能: Excel名簿生成、写真貼り付け、プロキャスデータ更新、バッチ処理
management/
おはよう和子さん (kazuko_departure_watch)
キャストの出発予定時間と起床予定時間を管理するLINE Bot連携システム。18時から22時の間に未登録者へリマインド通知を送信し、予定時間を過ぎても報告がない場合はTwilioで電話発信。Procastデータと連携して翌日出勤者のみに通知し、未登録者や緊急アラートを管制に通知する機能を備える。
主な機能: 出発・起床時間管理、LINE Bot連携、リマインド通知、電話発信、管制通知
kazuko_departure_watch/
Slack RAGシステム (slack_channel)
Slackチャンネルのメッセージデータをベクトルデータベースに保存し、セマンティック検索と質問応答を実現するRAG（Retrieval-Augmented Generation）システム。チャンネルメッセージをエンベディング化してベクトルDBに格納し、自然言語クエリで過去の会話を検索・参照できる。Webインターフェースを提供。
主な機能: ベクトルDB、セマンティック検索、質問応答、Web UI
slack_channel/
簡単な掲示板アプリ (simple-bulletin-board)
Next.js 16のApp Router、TypeScript、Tailwind CSS、SQLiteを使用したシンプルな掲示板アプリケーション。投稿の作成、時系列順の一覧表示、削除機能を提供。better-sqlite3を使用した軽量なデータベース構成で、開発環境と本番環境の両方に対応。
技術スタック: Next.js 16, TypeScript, Tailwind CSS, SQLite
simple-bulletin-board/
Slack Workflow Analysis (slack_workflow_analysis)
Slackワークフローのデータを分析するツール。大量のJSON形式のワークフローデータから情報を抽出し、ワークフローの構造、使用状況、パターンを分析。データの可視化とレポート生成機能を備える。
主な機能: ワークフロー抽出、データ分析、レポート生成
slack_workflow_analysis/
名簿管理くん (manager)
Google Sheetsをマスターデータとして、警備員名簿のExcelファイルを自動生成するFastAPIベースのシステム。Google Sheets APIから社員データを取得し、テンプレートExcelにデータを書き込み、写真を自動的に貼り付け。データ検証、一括名簿生成、統計情報取得、エクスポート機能（CSV/JSON）を提供。APIキー認証とレート制限に対応。
技術スタック: FastAPI, Python, Google Sheets API, openpyxl, Pillow
manager/
警備員名簿用DB更新プログラム (guardroster_db)
複数のExcelファイルからデータを読み取り、名前をキーとして警備員名簿用DB.xlsxのデータを更新するPythonプログラム。列マッピング設定による異なる構造のファイル対応、ドライラン機能、バックアップ自動作成、詳細なログ出力を備える。構造確認機能でファイルの整合性を事前にチェック可能。
主な機能: Excelデータ更新、列マッピング、ドライラン、バックアップ、ログ記録
guardroster_db/
Slack Channel Data Exporter
Slackチャンネルのメッセージ履歴と添付ファイルをエクスポートするCLIツール。メッセージをJSONL形式で保存し、ChatGPTなどのLLMで読みやすいMarkdown形式にも変換可能。複数チャンネルの同時エクスポート、期間指定、スレッド返信の取得に対応。Slack APIのレート制限を考慮した安全な処理を実装。
主な機能: メッセージエクスポート、ファイルダウンロード、JSONL/Markdown変換、期間指定
slack-channel-data-exporter/

仕様書指示書バリデータ
仕様書やClaudeCode向け指示書の品質を保証するバリデーションアプリケーション。矛盾・不整合の検出、用語の統一性チェック、自己完結性の評価、危険操作パターンの検出を行う。最小限の修正提案と監査耐性スコアの算出、改善提案の生成を提供。Spec TrackとInstruction Trackの2トラック制で多角的に品質を評価。
主な機能: 矛盾検出、用語統一チェック、危険操作検出、監査耐性スコア、改善提案

人間指示を起点に、ChatGPT → Claude → ChatGPT → ClaudeCode実行からなる思考サイクルを最大10回まで実行する制御アプリケーション。1人間指示＝1実行セッションとして、10回目の思考サイクルで必ず安全停止する設計。AI間のデータ受け渡しを管理し、実行履歴を記録。
主な機能: 思考サイクル制御、最大10回実行、安全停止、実行履歴管理

NewTOS v1.3 (newtos)
Takagi Flow準拠のジョブ制御システム。STOP/RESUME/TERMINATEによる人間判断を介したジョブ制御を実現。Slack通知・入力連携、Claude API統合、Run1/Run2/Run3の3段階進行制御を備える。AI間データ受け渡し機能、エラー分類、本番運用チェックリスト機能を提供。安全ゲート機構により、意図しない自動実行を防止。
主な機能: ジョブ制御、Slack連携、Claude API統合、3段階Run制御、エラー分類
newtos/
Indeed応募通知システム (recruit)
Indeedおよびジモティーからの求人応募メールを自動取得し、SlackとLINEに通知する自動化システム。GmailのIMAP経由で未読メールをチェックし、応募メールを自動判別。応募者名と応募内容確認URLを抽出して通知。Railway上で常時稼働し、test/prodモードによる環境切り替え機能を備える。
主な機能: メール監視、自動判別、Slack/LINE通知、常時稼働
recruit/

Takagi Flow準拠のオーケストレーターシステム。複数のステップを自動実行し、ログを管理するパイプライン制御ツール。各ステップのPythonスクリプトを順次実行し、実行結果を記録。エラーハンドリングとログファイル管理機能を提供。
主な機能: パイプライン実行、ステップ管理、ログ記録、エラーハンドリング
