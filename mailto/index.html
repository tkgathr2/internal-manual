📧 Eightでメールが開かないときの復旧手順

スタッフ向け簡易マニュアル

💡 原因：Eight のメールボタンを押しても反応しない場合、PC（Windows）の「mailto設定」が壊れているのが原因です。
以下の手順で必ず復旧できます。
1
Windows の「MAILTO」の既定アプリを Chrome に設定する
何をする？
メールのリンクをクリックしたとき、ChromeブラウザでGmailが開くようにする設定です。
  1. Windowsの「設定」を開く
    画面左下の スタートボタン（Windowsマーク） をクリック →
    ⚙️ 歯車アイコン（設定） をクリック
  2. 「アプリ」をクリック
    設定画面の左側メニューから「アプリ」を探してクリック
  3. 「既定のアプリ」をクリック
    アプリの画面で「既定のアプリ」を探してクリック
  4. 「リンクの種類で既定値を選択する」をクリック
    画面を下にスクロールすると見つかります
  5. 検索窓に「MAILTO」と入力
    上部の検索窓に MAILTO と入力して検索
  6. 「Google Chrome」を選んで「常に使う」をクリック
    MAILTOの項目をクリック → 一覧から Google Chrome を選択 → 「常に使う」
🔍 確認ポイント

✅ MAILTOの横に「Google Chrome」と表示されていればOK！

❌ 別のアプリが表示されている場合は、やり直してください

【Eightメール復旧】ステップ1で困っています。 現在やっていること： 「Windows の MAILTO の既定アプリを Chrome に設定する」 （メールリンクをクリックしたときにChromeでGmailが開くようにする設定） 手順： 1. 画面左下のスタートボタン → ⚙️歯車アイコン（設定）をクリック 2. 「アプリ」をクリック 3. 「既定のアプリ」をクリック 4. 「リンクの種類で既定値を選択する」をクリック 5. 検索窓に MAILTO と入力 6. Google Chrome を選んで「常に使う」をクリック 困っている内容： （ここに具体的に何がわからないか、どこでつまずいているかを書いてください）
2
Chrome の mailto ブロック設定を解除する
何をする？
Chromeが「Gmailでメールを開くこと」をブロックしていないか確認し、ブロックされていたら解除します。
  1. Chromeブラウザを開く
    デスクトップやタスクバーにある Chrome（丸い赤・黄・緑・青のアイコン）をクリック
  2. 以下のURLをChromeのアドレスバーにコピーして開く
    下の黒いボックスをクリックするとコピーできます。
    Chromeの上部にあるアドレスバー（URLが表示されているところ）に貼り付けてEnterキーを押してください。
chrome://settings/handlers
  1. 「プロトコルの処理を許可しないサイト」を確認
    画面を下にスクロールして、この項目を探してください
🔍 確認ポイント

❌ mail.google.com がある場合
　→ 右にある「✕」ボタンをクリックして削除してください

✅ mail.google.com がない場合
　→ そのままでOK！次のステップへ進んでください

【Eightメール復旧】ステップ2で困っています。 現在やっていること： 「Chrome の mailto ブロック設定を解除する」 （ChromeがGmailでメールを開くことをブロックしていないか確認する設定） 手順： 1. Chromeブラウザを開く 2. アドレスバーに「chrome://settings/handlers」をコピーして開く 3. 「プロトコルの処理を許可しないサイト」を確認 4. mail.google.com があれば「✕」で削除、なければそのままでOK 困っている内容： （ここに具体的に何がわからないか、どこでつまずいているかを書いてください） 例：「プロトコルの処理を許可しないサイト」という項目が見つからない、など
3
Gmail に mailto を許可する
⭐
これが一番重要なステップです！
何をする？
Gmailに「メールリンクを開く許可」を与えます。これをしないとGmailがメールリンクに反応しません。
  1. 下のボタンをクリックしてGmailを開く
👉 Gmail を開く（許可設定用）
  1. 画面上部にポップアップが表示されたら「許可する」をクリック
    「mail.google.com がメールリンクを開くことを許可しますか？」
    のようなメッセージが表示されます
  2. 「完了」をクリック
🔍 確認ポイント

✅ ポップアップで「許可する」→「完了」を押せればOK！

⚠️ ポップアップが自動で出ない場合

URLバー（アドレスバー）の右側に、ひし形◇が2個重なったようなアイコンがあります。
それをクリックすると、許可のポップアップが表示されます。

【Eightメール復旧】ステップ3で困っています。 現在やっていること： 「Gmail に mailto を許可する」（一番重要なステップ） （Gmailに「メールリンクを開く許可」を与える設定） 手順： 1. 以下のURLをクリックしてGmailを開く https://mail.google.com/mail/?extsrc=mailto&url=mailto:[email protected] 2. 画面上部にポップアップが表示されたら「許可する」をクリック 3. 「完了」をクリック 困っている内容： （ここに具体的に何がわからないか、どこでつまずいているかを書いてください） 例：ポップアップが出てこない、「許可する」ボタンが見つからない、など
4
Eightで最終確認
何をする？
実際にEightを開いて、メールボタンが正常に動くか確認します。
  1. 下のボタンをクリックしてEightを開く
📇 Eightを開く
  1. 名刺一覧から誰かの名刺を開く
    メールアドレスが登録されている名刺を選んでください
  2. メールアイコン（✉️）をクリック
    名刺の詳細画面にあるメールアイコンをクリック
🔍 確認ポイント

✅ Gmailの新規作成画面が開く
　→ 成功です！🎉 ステップ5へ進んでください

❌ 反応しない・Gmailが開かない
　→ ステップ1〜3をもう一度確認してください

5
完了報告メールを送信する
何をする？
設定が完了したことを髙木に報告します。下のボタンを押すと、報告メールが自動で作成されます。
  1. 下のボタンをクリック
    Gmailの新規作成画面が開き、宛先・件名・本文が自動で入力されます
🎉 完了報告メールを送信する
  1. そのまま「送信」ボタンをクリック
    内容を確認して、Gmailの送信ボタン（青いボタン）を押してください
🔍 確認ポイント

✅ メールを送信できれば、すべて完了です！お疲れ様でした！
